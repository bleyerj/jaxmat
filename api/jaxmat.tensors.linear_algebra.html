isotropic_function(fun, A)[source]#

Computes an isotropic function of a symmetric 3x3 matrix.

Parameters:

  • fun (callable) – A scalar function f(x)

  • A (jax.Array) – A symmetric 3x3 matrix

Returns:

A new 3x3 matrix such that f_A = sum_i f(lambda_i) n_i times n_i

Return type:

jax.Array
